# The-Legend-Of-Poopfly
Fett coolt, rougeaktigt, textbaserat äventyrsspel.

Generera en helt unik karaktär varje spelomgång och utforska en labyrint av slumpmässigt utvalda rum. Hitta skatter av olika kvaliteter, slåss mot monster och bossar och gå upp i nivå för att till slut kunna ta dig an den slutgiltiga utmaningen för att hitta skatten du kom hit för att hitta: Den enda kvalitet 0 skatten i spelet... poopfly!
